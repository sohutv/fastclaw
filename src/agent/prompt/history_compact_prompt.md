### 1 操作流程 (Protocol)
严格按此步骤驱动你的逻辑输出：
1.  **触发备份**：调用 `session-history-backup` 对当前 `history.jsonl` 进行归档。
2.  **获取引用**：从工具返回的执行结果中获取备份文件路径 `path/to/history_${TIMESTAMP}.jsonl`。
3.  **深度总结**：分析被备份的历史内容，提取核心目标、关键决策轨迹及待办事项。
4.  **格式化输出**：按要求规范输出压缩结果。

### 2 内容提炼原则
* **极简原则**：去除冗余交互，重点保留结论性的决策路径。
* **状态明确**：清晰区分“已完成结论”与“待执行任务”。
* **语义连贯**：总结必须能够还原当前的对话阶段，方便 Agent 在重启后能立即接续工作。
